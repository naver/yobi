FAQ
  • HIVE를 설치하고 싶어요.
    HIVE를 설치하기 전에 준비해야 할것은 다음과 같습니다.
    • JDK 1.6이상
    • Play Framework
    • Git Client
    위의 프로그램을 모두 인스톨 하신후 소스코드를 받습니다. 
    # git clone https://github.com/nforge/hive.git
    인스톨된 폴더로 가서 HIVE를 실행시키면 됩니다. 
    # play run
  • 프로젝트를 새로 생성하고 싶어요.

    상단의 New Project를 클릭하신후 필요한 정보를 입력하시면 됩니다.

    공개설정에서 공개를 택하게 되면 해당 프로젝트의 맴버가 아닌 사용자들도 해당 프로젝트를 둘러볼수 있게 되며 맴버가 아니라면 code 저장소를 익명으로 접근하여 소스코드를 받아 갈 수는 있지만 소스코드를 수정하지는 못합니다. 공개설정에서 비 공개를 택하게 되면 해당 프로젝트의 맴버가 아닌 사용자들은 단지 설명과 이름만을 볼수 있습니다.

    코드 저장소 방식은 현재 Git과 Subvision을 지원합니다. Subvision과 Git은 전세계적으로 코드관리 시스템으로 널리 쓰이고 있으며 충분한 신뢰성과 성능을 가지고 있습니다.

    위의 내용을 다 작성하셨다면 이용약관옆의 물음표를 눌러 약관을 살펴보시고 동의함을 눌러주시면 됩니다. 마지막으로 save버튼을 누르면 새로운 프로젝트를 생성하실수 있습니다.

  • 내가 참여하는 프로젝트들은 어디서 볼수 있나요?
    메인화면 우측 하단에 다음과 같이 참여하고 있는 프로젝트의 목록을 볼수 있습니다. 자물쇠가 있는 것은 비공개 프로젝트이며 자물쇠가 없는 것은 공개 프로젝트 입니다. 괄호 안은 프로젝트를 만든사람을 나타내며 괄호 앞은 프로젝트의 제목을 나타냅니다. 혹은 자신의 정보 페이지에서도 확인하실수 있습니다.
  • 프로젝트 탈퇴는 어떻게 하나요.
    혹은 자신의 정보 페이지에서 참여하고 있는 프로젝트 목록을 볼수있고 탈퇴도 할수 있습니다. 자신이 프로젝트의 유일한 관리자라면 해당 프로젝트에서 탈퇴를 할수 없습니다.
  • 게시판에서는 어떠한 것들을 할수 있나요?
    게시판에서는 다음과 같은 기능이 가능합니다.
    • 게시물 상세 조회: 사용자는 게시물의 내용을 볼 수 있다.
    • 게시물 댓글 등록: 로그인 유저는 게시물에 댓글을 남길 수 있다.
    • 게시물 댓글 조회: 사용자는 게시물의 댓글을 볼 수 있다.
    • 게시물 댓글 삭제: 로그인 유저는 자신이 남긴 댓글을 삭제할 수 있다.
    • 관리자 게시물 댓글 삭제: 프로젝트 관리자는 댓글을 삭제할 수 있다.
    • 관리자 게시물 수정: 프로젝트 관리자는 게시물을 편집/삭제 할 수 있다.
  • HIVE의 버그를 발견했어요.
    HIVE는 현재 Open Source로 진행되고 있습니다. 버그를 발견하셨다면 HIVE 이슈트래커에 등록해 주시거나 패치를 만들어 보내주시면 됩니다.
}
